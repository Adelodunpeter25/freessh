# Welcome to FreeSH

FreeSH is a free, open-source, local-first SSH client built for developers and system administrators who need a reliable, feature-rich tool for managing remote servers and transferring files.

## What is FreeSH?

FreeSH combines the functionality of a terminal emulator, SSH client, and SFTP file manager into a single, intuitive application. Whether you're managing multiple servers, transferring files between remote locations, or executing commands across your infrastructure, FreeSH provides the tools you need.

## Key Features

### Terminal Management
- **Multiple Sessions**: Open and manage multiple SSH connections simultaneously with tabbed interface
- **Local Terminal**: Built-in local terminal for quick command execution
- **Terminal Search**: Search through terminal output to find specific commands or results
- **Session Persistence**: Your connections and settings are saved between sessions

### Secure Authentication
- **Multiple Auth Methods**: Support for password and SSH key authentication
- **SSH Key Management**: Browse and select SSH keys with built-in file picker
- **System Keychain Integration**: Securely store credentials using your system's keychain
- **Known Hosts Management**: Review and confirm host keys before adding to trusted hosts

### Advanced SFTP
- **Dual-Panel Interface**: Side-by-side file browsing for easy transfers
- **Remote-to-Remote Transfers**: Transfer files directly between servers without downloading locally
- **Drag and Drop**: Intuitive drag-and-drop support for all transfer types
- **Bulk Operations**: Select and transfer multiple files and directories at once
- **Progress Tracking**: Real-time progress monitoring with cancellation support
- **Directory Support**: Recursive transfers for entire directory structures

### Port Forwarding
- **Local Forwarding**: Forward local ports to remote destinations
- **Remote Forwarding**: Forward remote ports to local destinations
- **Dynamic Forwarding**: SOCKS proxy support for secure browsing
- **Auto-Start**: Configure tunnels to start automatically on connection
- **Tunnel Management**: Easy-to-use interface for managing all your port forwards

### Productivity Features
- **Command Snippets**: Save and organize frequently used commands
- **Command History**: Track and reuse commands across sessions
- **Variables & Placeholders**: Create dynamic snippets with runtime input
- **Connection Groups**: Organize connections into folders and groups
- **Search & Filter**: Quickly find files and connections
- **Theme Support**: Light, dark, and system theme options

### Performance & Reliability
- **Auto-Reconnect**: Automatic reconnection with exponential backoff
- **Concurrent Transfers**: Multiple file transfers running in parallel
- **Efficient Streaming**: Direct server-to-server transfers without local storage
- **Memory Optimized**: Efficient handling of large files and directories

## Who is FreeSH For?

FreeSH is designed for:

- **DevOps Engineers** managing multiple servers and deployments
- **System Administrators** performing routine maintenance and file transfers
- **Developers** working with remote development environments
- **Security Professionals** requiring secure, auditable remote access
- **Anyone** who needs a reliable SSH and SFTP client

## Platform Support

FreeSH is built with Electron and Go, providing native performance across:
- macOS (Intel and Apple Silicon)
- Windows
- Linux
